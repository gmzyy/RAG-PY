# STAICKA - Sistema RAG de Análisis de Documentos 🚀

Este es un motor de búsqueda y análisis semántico diseñado para procesar archivos PDF (como CVs o manuales corporativos) y extraer información clave de forma inteligente. Desarrollado como parte del ecosistema de soluciones de IA de **STAICKA**.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Python 3.10+
* **Orquestación:** LangChain
* **Base de Datos Vectorial:** ChromaDB
* **Modelos de IA:** Google Gemini (Generación) y Text-Embeddings (Vectorización)
* **Backend Sugerido:** FastAPI / MongoDB

## ✨ Características Principales
* **Ingesta Inteligente:** Procesa múltiples documentos PDF y los convierte en vectores matemáticos.
* **Búsqueda Semántica:** No solo busca palabras clave, entiende el contexto de la consulta.
* **Privacidad de Datos:** Configurable para ejecución local o híbrida.
* **Ranking de Perfiles:** Ideal para la revisión automatizada de CVs y selección de talento.
